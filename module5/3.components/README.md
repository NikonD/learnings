# 3.components — учебный пример по React‑компонентам

Небольшой **отдельный React‑проект** (Vite), который показывает, как:

- делить интерфейс на **контейнерные** и **презентационные** компоненты;
- выносить переиспользуемые элементы в `ui/`;
- собирать экран из маленьких, понятных частей.

## Структура

- `src/ui` — общие элементы оформления и каркас страницы:
  - `App.jsx` — точка входа, собирает демо‑блоки;
  - `Layout.jsx` — общий лейаут страницы;
  - `Button.jsx`, `Card.jsx`, `styles.css` — переиспользуемые UI‑компоненты.
- `src/users` — пример «контейнер + презентационный компонент»:
  - `UserListContainer.jsx` — хранит состояние и логику;
  - `UserCard.jsx` — только отображает данные и дергает колбэки.
- `src/counter` — пример разбиения одного виджета на несколько мелких компонентов:
  - `CounterCard.jsx` (внутри — `CounterValue` и `CounterControls`).

## Как запускать

```bash
cd module5/3.components
npm install
npm run dev
```

Откроется обычное React‑приложение на Vite (по умолчанию `http://localhost:5173`), в котором видно:

- пример контейнерного компонента с «тупыми» карточками;
- пример разбиения одного виджета (счётчик) на несколько переиспользуемых частей.
