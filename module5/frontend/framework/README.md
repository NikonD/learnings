# Company API — React (учебный фронт)

Клиент на **React** (Vite) для бэкенда **module3/company_api_with_orm**.

## Запуск

1. Запустите бэкенд: `cd module3/company_api_with_orm` → `docker-compose up -d` → `python main.py`
2. Установите зависимости и запустите dev-сервер:

```bash
npm install
npm run dev
```

3. Откройте в браузере адрес из терминала (обычно http://localhost:5173).

## Сборка

```bash
npm run build
```

Статика будет в папке `dist/`.
